{% blocktrans %} There are two ways to search for data. {% endblocktrans %}

{% trans 'Search Bar' %}

{% blocktrans %} You can use the search bar on the Bus Open Data digital platform homepage. The search bar function looks for specific words in the title and description in each data set. Search terms can be geographic areas or bus operator names. By default, best matches are returned first. However, you can also use the filter function on the right hand side of the return page to filter out any terms you do not want to see. {% endblocktrans %}

{% trans 'API Searching' %}

{% blocktrans %} Each data set has an API key , which can be copied from the data page. The Bus Open Data API provided by JSON and extensions. {% endblocktrans %}
