{% trans 'What is TransXchange?' %}

{% blocktrans %} TransXchange is the UK nationwide standard for exchanging bus schedules and related data. The TransXchange website provides a brief overview on TransXchange and directions to the TransXchange 2.1 schema. The TransXchange Schema Guide is an electronic document that explains how to use TransXchange. This guide has content suitable for both users and implementers. It provides an overview of how the TransXchange model works, a detailed description of the schema element, worked examples and technical annexes, including a specification of the integrity rules used in TransXchange. {% endblocktrans %}
