{% trans 'Browse for specific data' %}

{% blocktrans %} Publishers supply data sets and data feeds to the Bus Open Data Service. You can view, download or feedback on these individually when you {% endblocktrans %} {% trans 'Browse for specific data' %} {% blocktrans %} and click on the data to find its detail page. {% endblocktrans %}

{% trans 'Data detail page' %}

{% blocktrans %} Each data set/feed supplied to the service has a detail page, where you can view: {% endblocktrans %}

  • Metadata
  • Changelog
  • API call with that data set ID
  • Feedback box
  • Subscription options
  • Link to underlying data for static data types

{% trans 'Feedback' %}

{% blocktrans %} Feedback will be sent via email to the data publisher, enhancing the feedback loop between developers and publishers, in turn improving the overall data quality. {% endblocktrans %}

{% trans 'Subscribe' %}

{% blocktrans %} In the detail page you can subscribe to data sets/feeds so you will receive email notifications if the publisher makes any changes. {% endblocktrans %}

{% trans 'Search bar functionality' %}

{% blocktrans %} The search bar matches terms entered against metadata such as the title, description or data ID. {% endblocktrans %}
