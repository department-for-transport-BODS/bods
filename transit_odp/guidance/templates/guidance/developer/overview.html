{% trans 'Overview' %}

{% blocktrans %} The Bus Open Data Service provides timetables, bus location and fares data for local bus services across England. {% endblocktrans %}

{% blocktrans %} The Bus Services Act 2017 requires all operators of local bus services in England to provide this information. The transition period is from 2020-2021. {% endblocktrans %}

{% blocktrans %} The Bus Open Data Service is freely available for anyone to use, providing a single location to discover and access data. As well as provide feedback to data publishers. You do not need a license to use the data. {% endblocktrans %}
