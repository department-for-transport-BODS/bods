{% trans 'Timetables data support' %}

{% blocktrans %} Local transport authorities may support operators by sending them the digitised TranXChange files made from their paper registration. The operator can then publish it on the service. {% endblocktrans %}

{% blocktrans %} A number of smaller bus operators may wish to post bus information to you, as many are currently doing whilst registering a service. You may offer to send these files to operators or support them in producing their TransXChange files for the service. {% endblocktrans %}

{% blocktrans %} Methods of producing TransXChange 2.4 include procuring scheduling software or using the free TxC 2.4 tool provided by DfT, from the following website: {% endblocktrans %}

https://www.gov.uk/guidance/publish-bus-open-data

{% blocktrans %} If you require additional support, DfT can support the creation of TxC using the tool. For more information please contact {% endblocktrans %} bodshelpdesk@kpmg.co.uk

{% blocktrans %} Please ensure the timetables data provided aligns with TransXChange version 2.4. {% endblocktrans %}

{% trans 'Bus location data support' %}

{% blocktrans %} A limited number of operators still do not have ETMs that can provide SIRI-VM data which aligns with the Department for Transport SIRI-VM Profile. This also enables crowding information to be provided to the service. {% endblocktrans %}

{% blocktrans %} Please ensure ETM suppliers procured can provide bus location data in feeds that align with the standards documented in the {% endblocktrans %} operator requirements. {% blocktrans %} Once a supplier has been procured, operators may require support getting set up. {% endblocktrans %}

{% trans 'Fares data support' %}

{% blocktrans %} Operators who have ETM suppliers can talk with them about hosted NeTEx solutions. A large proportion of fares data already resides within ETM systems. This can be leveraged to provide NeTEx to BODS. There may be a need for additional product development, so ensure your needs are understood by your supplier. {% endblocktrans %}

{% blocktrans %} Alternatively, you can support operators using the free Fare Data Build Tool to produce NeTEx which can be found here: {% endblocktrans %}

https://fares-data.dft.gov.uk/

{% blocktrans %} Please ensure fares data provided aligns to the standards in the {% endblocktrans %} operator requirements.
